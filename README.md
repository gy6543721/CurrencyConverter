# CurrencyConverter

Native Android Application written in Kotlin with Jetpack Compose.  
Implement with MVVM structure, support Dark Mode.  
[Demo Video](https://youtu.be/O8s3Bd3pNNM)  
<br>

# Main Libraries Used
* Jetpack Compose
* Coroutines
* Room
* Dagger Hilt
* Retrofit
* OkHttp3
* Gson
* Junit
